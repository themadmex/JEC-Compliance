# Jean Edwards Design System

## Theme Overview

The Jean Edwards platform features a sophisticated, modern, and professional aesthetic. It blends traditional trust signals (serif typography, navy/dark green colors) with modern digital product design to create a premium enterprise feel.

### Color Palette

- **Primary Text:** Dark Charcoal (`#1A1A1A`)
- **Primary Background (Light Mode):** Soft cream / Off-white (`#FCFCF9` or `#F9F9F6`)
- **Sidebar Background:** Deep Slate Blue / Navy (`#162132` or `#1A2436`) or Dark Forest Green depending on the section.
- **Accent Color:** Warm Gold / Mustard (`#C6A45C` or `#D4AF37`) - used for active navigation states, progress bars, buttons, and icons.
- **Secondary Text/Muted:** Medium gray for secondary information (`#666666`).
- **Semantic Colors:**
  - **Success/Pass:** Golden/Mustard (in progress bars) or standard green text.
  - **Warning:** Amber/Yellow text.
  - **Danger/Fail:** Deep Red/Burgundy (`#8B0000`) text.

### Typography

- **Headings (Brand/Primary):** Elegant modern Serif font (e.g., Playfair Display, Merriweather, standard Georgia). Used for the brand name, page titles (e.g., "Compliance progress"), and major metrics.
- **Body & UI Elements:** Clean Sans-Serif font (e.g., Inter, Roboto, Helvetica). Used for navigation, tabular data, standard text, and secondary labels.

### Layout Principles

- **Navigation:** Persistent left-side vertical sidebar. Active state is highlighted with a slight background lightening and a left border indicator in the accent gold. Icons accompany labels.
- **Content Area:** Spacious layout with generous padding. Uses a card-based structure.
- **Cards:** White backgrounds (`#FFFFFF`) with subtle, very soft drop shadows or delicate borders, housing distinct sections of information. Corners are slightly rounded (e.g., `8px` or `12px` border-radius).

## Key Components

- **Progress Bars:** Distinctive styling - gold fill with a light beige track, rounded ends. Often accompanied by large serif percentage metrics.
- **Buttons:**
  - Primary: Solid colors (Dark Green or Gold), rounded corners, clear contrast text.
  - Secondary: Outline style with text color matching the border.
- **Data Tables:** Clean rows, no vertical dividers, subtle horizontal dividers. Column headers are slightly bolder sans-serif. Status badges or colored text are used for clear state indication (Passed, Failed, Warning).
- **Cards/Widgets:** Used to display summary information (e.g., "Policies", "Tests", "Vendors") and feature blocks (e.g., "Trust Center" banner with a background image or dark overlay).
